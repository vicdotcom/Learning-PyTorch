# Learning-PyTorch
This repository covers a structured study of PyTorch fundamentals with the long-term goal of applying PyTorch in deep learning applications. Follows the [PyTorch for Deep Learning & Machine Learning](https://www.youtube.com/watch?v=V_xro1bcAuA&t=2598s) curriculum.

## Content Covered
| Topic | Folder | Key Concepts |
|-------|--------|--------------|
| PyTorch Fundamentals | `01_tensors_basics/` | Tensor creation, indexing, broadcasting |
| Autograd | `02_autograd/` | Gradient tracking, backward pass |
| Neural Networks | `03_neural_networks/` | nn.Module, forward pass |
| Training Loop | `04_training_loop/` | Loss functions, optimizers |
| GPU Training | `05_gpu_training/` | CUDA, device management |
